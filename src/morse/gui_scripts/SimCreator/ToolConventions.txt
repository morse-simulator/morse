Tools
* Each tool has to be in the 'Tools' folder.
* Each tool has to be in a folder that has the same name as the tool.
* The .blend file containing the tool has to have [name].blend as filename, where [name] is the
  name of the tool
* All the objects that make up the tool have to be added to a group that has the same name as the tool.
* A tool must have an empty that is called 'AnchorC', this is the place where the tool will be connected
  to an anchorPoint, all objects that make up the tool have to be parented to this AnchorC.
* Anchor points on a tool must be an empty called 'AnchorP_{unique name}', where {unique name} is a name
  that has to be different from the other AnchorPoints.
* tool scale has to be 1 blender unit = 1 meter.
